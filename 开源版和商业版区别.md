## **开源版和商业版区别**



| 特性                    | 开源版本   | 商业版本  |
| ----------------------- | ---------- | --------- |
| agent运行环境           | 依赖JDK1.8 | 无依赖    |
| agent实现方式           | java       | go        |
| 端口检测                | 无         | 有        |
| 进程名称监控            | 无         | 有        |
| CPU温度                 | 无         | 有        |
| 磁盘IO                  | 无         | 有        |
| docker监控              | 无         | 有        |
| 集成钉钉微信等告警方式  | 无         | 有        |
| agent上报间隔时间       | 120s       | 30s，可调 |
| 日志文件检测            | 无         | 有        |
| 主机画像                | 无         | 有        |
| 大屏可视化              | 无         | 有        |
| 系统信息                | 有         | 有        |
| 网络拓扑图              | 无         | 有        |
| 资产管理                | 无         | 有        |
| cpu使用率               | 有         | 有        |
| 内存使用率              | 有         | 有        |
| 网络流量                | 有         | 有        |
| 系统负载                | 有         | 有        |
| 磁盘空间                | 有         | 有        |
| 进程管理                | 有         | 有        |
| 告警开关组              | 无         | 有        |
| 堡垒机(跳转机)、web ssh | 无         | 有        |
| 磁盘smart检测           | 无         | 有        |
| 批量执行下发指令        | 无         | 有        |
